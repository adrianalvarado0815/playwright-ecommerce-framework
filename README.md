# Playwright E-commerce Framework

## Objective

Build a professional automation framework using Playwright and TypeScript.

## Technologies

- Playwright
- TypeScript
- Node.js

## Project Structure

- tests
- pages
- fixtures
- utils

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```
